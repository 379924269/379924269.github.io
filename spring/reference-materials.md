# 参考资料

以下是一些学习 Spring Cloud Alibaba 的优质参考资料：

## 官方文档

- **[Spring Cloud 官方文档](https://spring.io/projects/spring-cloud)**
  Spring Cloud 的官方文档，包含了详细的使用指南和API文档。

- **[Spring Cloud Alibaba 官方文档](https://spring.io/projects/spring-cloud-alibaba)**
  Spring Cloud Alibaba 子项目的官方文档，包含了 Nacos、Sentinel、RocketMQ、Dubbo、Seata 等组件的详细说明。

- **[Nacos 官方文档](https://nacos.io/zh-cn/docs/quick-start.html)**
  Nacos 的官方文档，详细介绍了 Nacos 的安装、配置和使用。

- **[Sentinel 官方文档](https://sentinelguard.io/zh-cn/docs/introduction.html)**
  Sentinel 的官方文档，详细介绍了 Sentinel 的安装、配置和使用。

- **[RocketMQ 官方文档](https://rocketmq.apache.org/docs/quick-start/)**
  RocketMQ 的官方文档，详细介绍了 RocketMQ 的安装、配置和使用。

- **[Dubbo 官方文档](https://dubbo.apache.org/zh/docs3-v2/)**
  Dubbo 的官方文档，详细介绍了 Dubbo 的安装、配置和使用。

- **[Seata 官方文档](https://seata.io/zh-cn/docs/overview/what-is-seata.html)**
  Seata 的官方文档，详细介绍了 Seata 的安装、配置和使用。

## 书籍

- **[Spring Cloud Alibaba 微服务实战](https://book.douban.com/subject/35129788/)**
  作者：方志朋，详细介绍了 Spring Cloud Alibaba 的核心组件和使用方法。

- **[Spring Cloud 微服务架构开发实战](https://book.douban.com/subject/30395302/)**
  作者：周立，从实战角度出发，介绍了 Spring Cloud 的各种组件和最佳实践。

- **[Spring Boot 与 Spring Cloud 微服务开发](https://book.douban.com/subject/34909675/)**
  作者：王福强，结合 Spring Boot 和 Spring Cloud，介绍了微服务开发的完整流程。

## 工具和插件

- **[Spring Initializr](https://start.spring.io/)**
  Spring 官方提供的项目初始化工具，可以快速创建 Spring Boot 和 Spring Cloud 项目。

- **[Spring Tool Suite](https://spring.io/tools)**
  Spring 官方提供的 IDE 插件，包含了对 Spring Boot 和 Spring Cloud 的支持。

- **[IntelliJ IDEA Spring Assistant](https://plugins.jetbrains.com/plugin/10229-spring-assistant)**
  IntelliJ IDEA 的 Spring 助手插件，提供了对 Spring Boot 和 Spring Cloud 的支持。

- **[Nacos Docker](https://github.com/nacos-group/nacos-docker)**
  Nacos 的 Docker 镜像，便于快速部署 Nacos 服务。

- **[Sentinel Dashboard](https://github.com/alibaba/Sentinel/tree/master/sentinel-dashboard)**
  Sentinel 的 Dashboard 工具，用于监控和管理 Sentinel 规则。
