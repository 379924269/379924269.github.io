# 学习路径

Spring Cloud Alibaba 的学习需要按照一定的顺序进行，以下是推荐的学习路径：

## 1. Spring Boot 基础

Spring Cloud Alibaba 基于 Spring Boot 构建，因此首先需要掌握 Spring Boot 的基础知识：

- Spring Boot 核心概念和自动配置
- Spring Boot 项目的创建和运行
- Spring Boot 中的依赖管理
- Spring Boot 中的常用注解
- Spring Boot 中的配置管理

## 2. 服务注册与发现 (Nacos)

Nacos 是 Spring Cloud Alibaba 中的服务注册与发现组件，需要学习：

- Nacos 服务端的搭建和配置
- Nacos 客户端的注册和发现
- Nacos 集群的搭建和高可用配置
- Nacos 的服务管理功能

## 3. 配置中心 (Nacos Config)

Nacos 同时也是一个配置中心，需要学习：

- Nacos Config 的基本配置和使用
- 配置的集中管理和版本控制
- 配置的动态刷新
- 配置的环境隔离

## 4. 流量控制与熔断降级 (Sentinel)

Sentinel 是 Spring Cloud Alibaba 中的流量控制和熔断降级组件，需要学习：

- Sentinel 的基本配置和使用
- 流量控制规则的配置
- 熔断降级规则的配置
- Sentinel Dashboard 的使用
- 系统保护规则的配置

## 5. 消息中间件 (RocketMQ)

RocketMQ 是 Spring Cloud Alibaba 中的消息中间件，需要学习：

- RocketMQ 的基本概念和架构
- RocketMQ 的安装和配置
- 消息的发送和消费
- 事务消息的使用
- 消息的顺序消费

## 6. RPC 框架 (Dubbo)

Dubbo 是 Spring Cloud Alibaba 中的 RPC 框架，需要学习：

- Dubbo 的基本概念和架构
- Dubbo 的服务定义和实现
- Dubbo 的服务注册与发现
- Dubbo 的负载均衡策略
- Dubbo 的服务监控

## 7. 分布式事务 (Seata)

Seata 是 Spring Cloud Alibaba 中的分布式事务解决方案，需要学习：

- Seata 的基本概念和架构
- AT 模式的使用
- TCC 模式的使用
- SAGA 模式的使用
- XA 模式的使用

## 8. API 网关 (Spring Cloud Gateway)

Spring Cloud Gateway 是 Spring Cloud 中的 API 网关组件，需要学习：

- Spring Cloud Gateway 的基本配置和使用
- 路由规则的配置
- 过滤器的实现
- 限流和熔断的配置
- 高可用部署

## 9. 监控与运维 (Spring Boot Admin)

Spring Boot Admin 是用于管理和监控 Spring Boot 应用的工具，需要学习：

- Spring Boot Admin Server 的搭建和配置
- Spring Boot Admin Client 的集成
- 应用监控和健康检查
- 日志查看和分析
- 环境信息和 JVM 监控

## 10. 微服务部署与容器化

微服务的部署和容器化是生产环境中的重要环节，需要学习：

- Docker 容器化部署
- Kubernetes 集群管理
- Helm 包管理
- CI/CD 流水线配置
- 服务网格 (Istio) 的使用
