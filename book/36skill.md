# 📖36计

## 胜战计

### 概述
我强敌弱,以最小的代价扩大胜势

| 计策 | 原文解释 | 现代应用 | 典型案例 |
|------|----------|----------|----------|
| **瞒天过海** | 防备周全时更容易麻痹大意；习以为常的事情往往就不会怀疑了 | **商业**：营销策略中的渐进式创新；**职场**：逐步改变工作流程 | 三国·吕蒙白衣渡江奇袭荆州 |
| **围魏救赵** | 攻击敌人必救的地方，使其退兵回救，从而解我方之围 | **商业**：攻击竞争对手的核心市场；**技术**：通过攻击系统薄弱环节分散注意力 | 战国·孙膑围魏救赵 |
| **借刀杀人** | 利用别人的力量去攻击敌人，自己不直接出手 | **职场**：借助他人力量完成目标；**商业**：利用第三方渠道推广 | 春秋·晏子二桃杀三士 |
| **以逸待劳** | 养精蓄锐，等待敌人疲劳，然后出击 | **技术**：缓存策略，避免重复计算；**商业**：等待市场时机成熟 | 战国·孙膑马陵之战 |
| **趁火打劫** | 趁敌人有危机时进行攻击获取利益 | **商业**：竞争对手危机时抢占市场；**投资**：市场低迷时抄底 | 春秋·勾践趁吴国饥荒伐吴 |
| **声东击西** | 表面上攻打东边，实际上攻打西边，制造假象迷惑敌人 | **市场营销**：佯攻一个领域，实际在另一个领域发力；**产品开发**：多线并行，重点突破 | 楚汉·韩信暗度陈仓（类似） |

## 敌战计

|------|----------|----------|----------|
| **无中生有** | 从没有中创造出有来，即制造假象迷惑敌人 | **产品**：概念产品验证市场；**营销**：制造稀缺感和期待感 | 战国·张仪离间齐楚 |
| **暗度陈仓** | 正面迷惑敌人，暗中从侧面进行突然袭击 | **商业**：明面上做传统业务，暗地里布局新赛道；**技术**：灰度发布 | 楚汉·韩信明修栈道暗度陈仓 |

| **笑里藏刀** | 外表和善，内心险恶；表面温和，实际暗藏杀机 | **商业**：合作中的风险防范；**职场**：警惕表面和善的竞争对手 | 战国·商鞅诱骗公子卬 |
| **李代桃僵** | 李树代替桃树枯死，比喻以此代彼或代人受过 | **商业**：牺牲局部利益保全整体；**项目管理**：优先级取舍 | 春秋·赵氏孤儿 |

## 攻战计

| 计策 | 原文解释 | 现代应用 | 典型案例 |
|------|----------|----------|----------|
| **打草惊蛇** | 打草惊动了草里的蛇，比喻做事泄密，惊动对手 | **产品**：小规模测试市场反应；**调研**：通过间接手段了解信息 | 春秋·秦穆公伐郑被发觉 |
| **借尸还魂** | 比喻已经死亡的事物，假借别的名义重新出现 | **商业**：老旧品牌重新包装；**技术**：旧技术在新场景的应用 | 三国·刘备借荆州 |

| **欲擒故纵** | 想要捉住他，故意先放开他，使他放松戒备 | **产品**：先免费后收费；**谈判**：先让步后争取 | 三国·诸葛亮七擒孟获 |

## 36计心法

### 核心原则
1. **实事求是**：计策是手段，不是目的，需根据实际情况灵活运用
2. **审时度势**：形势判断是前提，准确判断敌我态势
3. **见机行事**：把握时机，随机应变，不要拘泥于固定套路
4. **适度而为**：计策不能过度使用，过犹不及
5. **道德底线**：在商业和生活中运用，不能违背法律和基本道德

### 应用层次
| 层次 | 说明 |
|------|------|
| **技术层面** | 具体问题的解决方法 |
| **战术层面** | 局部竞争中的策略应用 |
| **战略层面** | 整体布局和长远规划 |
| **哲学层面** | 对人性和规律的深刻理解 |

### 古今对比
- **古代**：军事战争，你死我活的零和博弈
- **现代**：商业竞争、职场发展、人际关系，多是竞合关系
- **核心差异**：古代以消灭为目的，现代以共赢为目标
